# App Store Badges

Diese Platzhalter-SVGs sollten durch die offiziellen Apple App Store Badges ersetzt werden.

## So ersetzen Sie die Badges:

1. Die offiziellen Apple App Store Badges finden Sie auf Ihrem Mac unter:
   - `/Users/cem/Downloads/Download-on-the-App-Store/`

2. Kopieren Sie die SVG-Dateien aus den entsprechenden Sprachordnern:
   - **Deutsch**: `DE/Download_on_App_Store/Black_lockup/SVG/Download_on_the_App_Store_Badge_DE_RGB_blk_092917.svg`
   - **Englisch**: `US/Download_on_the_App_Store/Black_lockup/SVG/Download_on_the_App_Store_Badge_US-UK_RGB_blk_092917.svg`
   - **Französisch**: `FR/Download_on_the_App_Store/Black_lockup/SVG/Download_on_the_App_Store_Badge_FR_RGB_blk_092917.svg`

3. Benennen Sie die Dateien um zu:
   - `app-store-de.svg`
   - `app-store-en.svg`
   - `app-store-fr.svg`

4. Ersetzen Sie die aktuellen Platzhalter-Dateien in diesem Ordner.

## Alternative:

Sie können die offiziellen Badges auch direkt von Apple herunterladen:
https://developer.apple.com/app-store/marketing/guidelines/#downloadable-materials
